Automatic Bid Record

Bids are stored automatically in the extension database (IndexedDB · AutomaticBidRecord · store bids) when you finish an application (form submit with a matching control, or matching click on SPA flows). Extra button labels (one per line): a save runs when the control text contains one of these phrases (case-insensitive), in addition to built-in Apply / Submit application patterns.

To enable SQLite mirroring, run native\\install-windows.ps1 once and paste your extension ID when prompted. The DB file will be created at %LOCALAPPDATA%\\AutomaticBidRecord\\bids.sqlite.

Open the extension popup to view history and export Excel. Load the extension from chrome://extensions (Developer mode → Load unpacked).
